# Magnetic Resonance Network (MRN)

## แนวคิด
  เครือข่ายนี้ใช้สนามแม่เหล็กและเรโซแนนซ์เป็นสื่อกลางในการสื่อสาร โดยเปลี่ยนรูปแบบและความถี่ของสนามแม่เหล็กเพื่อส่งข้อมูล ทำให้สามารถทะลุผ่านสิ่งกีดขวางได้ เช่น ใต้น้ำ ใต้ดิน หรือผนังหนา

## จุดเด่น
  จุดเด่นคือไม่จำเป็นต้องมีเสา สายสัญญาณ หรือโครงสร้างอินเทอร์เน็ตหลัก เนื่องจากแต่ละโหนดสามารถสร้าง รับ และขยายสนามแม่เหล็กได้ด้วยตนเอง โครงสร้างเป็นแบบกระจายศูนย์ โดยทุกโหนดมีสถานะเท่าเทียมกันและช่วยพยุงเสถียรภาพของเครือข่าย ส่งผลให้ระบบมีความทนทานต่อความเสียหายบางส่วน และเหมาะกับสภาพแวดล้อมที่คลื่นวิทยุถูกรบกวน
