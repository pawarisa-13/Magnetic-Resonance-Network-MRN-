## How to Run the Mockup 🛠

To run this simulation, you need to open **two separate Terminal windows**—one for the Backend API and one for the Frontend UI.

### Step 1: Start the Backend (API & Logs)
Open your first Terminal and run:
```bash
cd /mrn-web-mockup/backend
npm run start:dev
```
*This will start the server on `http://localhost:3001` and display a live stream of backend routing logs.*

### Step 2: Start the Frontend (User Interface)
Open your second Terminal and run:
```bash
cd /mrn-web-mockup/frontend
npm run dev
```
*This will start the web application. Open [http://localhost:5173](http://localhost:5173) in your browser.*

---
**Tip for Presentation:** Keep the Backend terminal visible on the side of your screen while using the Frontend UI to show the real-time "Under the Hood" event logs to your professor!
